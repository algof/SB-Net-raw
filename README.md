# SB-Net-raw

End-to-end dataset preparation for CTU-13, NCC, and NCC-2 network flow data, plus a combiner for unified training sets.

- Code Ocean usage, data layout (sample/full), and run instructions: see `README-CO.md`.
- Quick start locally:
  - `pip install -r requirements.txt`
  - `python scripts/run_all.py --check`
  - `python scripts/run_all.py --datasets all`
